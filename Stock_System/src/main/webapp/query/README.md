# Stock Searching
https://alexhyman.github.io/StockSearching/

### 说明：
股票查询 (支持上证，深证，港股，美股查询)


### 实现功能：
1.输入上证，深证，港股或美股股票编号，即可显示最新股票信息  
2.每个股票市场提供了九支股票，点击图片即可显示最新股票信息，点击 “全部显示” 按钮即可全部显示
